=== Overleaf Upload Instructions ===

1. Go to https://www.overleaf.com
2. Click "New Project" → "Upload Project"
3. Upload this entire folder as a .zip
   OR create a new project and upload these files individually

Files included:
  - paper.tex              (main LaTeX file)
  - references.bib         (bibliography)
  - efficiency_comparison.png  (Figure 1)

OR use the "Elsevier CAS Template" in Overleaf and replace
the template content with paper.tex content.

Compile with: pdfLaTeX → BibTeX → pdfLaTeX → pdfLaTeX
